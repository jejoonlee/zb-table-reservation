## 매장 이용자



#### 매장 검색 기능

- 키워드로 매장을 찾을 수 있게 리스트로 보여준다
- 리스트를 통해 매장 디테일로 들어갈 수 있게 해준다



#### 매장 예약

- 매장 운영 시간에만 예약을 할 수 있음 (쉬는 시간도 고려)
- 예약을 할 때에 이미 지난 시간은 예약을 못 함
- **10분 전까지 키오스크를 통해서 방문 확인을 하라는 메세지 전달**



#### 키오스크, 10분전 방문 확인

- 10분 전에 방문을 못 했다면, 예약을 자동으로 취소
